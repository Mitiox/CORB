# CORB Image Editor

A sleek, modern web-based image editing tool built with React and Tailwind CSS, featuring a unique neomorphic design system.

## 🌟 Features

*   **Image Import:** Drag and drop images directly into the workspace or load them instantly via URL.
*   **Smart Cropping:** Free-form cropping or use preset aspect ratios (1:1, 16:9, 4:3, 3:2) for perfect framing.
*   **Color Picker (Eyedropper):** Built-in eyedropper tool to seamlessly pick and extract exact colors from your canvas.
*   **Background Management:** 
    *   Instantly remove or isolate image backgrounds.
    *   Apply solid background colors using a color wheel or exact hex inputs.
    *   Toggle transparent background grids.
*   **Workspace Controls:** Interactive zoom controls (+ / - / Reset) and panning for pixel-perfect adjustments.
*   **Flexible Exports:** Export your edited masterpiece as PNG, JPEG, or WEBP with adjustable quality sliders.
*   **Neomorphic UI:** A carefully crafted, tactile user interface with raised (convex) buttons, pressed input fields, and clean typography.

## 🛠️ Tech Stack

*   **Frontend:** React 18, TypeScript, Vite
*   **Styling:** Tailwind CSS (Custom Neomorphic theme)
*   **Image Manipulation:** `react-image-crop`
